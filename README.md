# manual.praisenter.github.io
Repo for the Praisenter application user manual.  The static output of this site is bundled with the 
application for offline help and deployed for online help.

## VS Code Setup
<kbd>Ctrl</kbd>+<kbd>Shift</kbd>+<kbd>P</kbd>
git clone https://github.com/praisenter/manual.praisenter.github.io.git
Install Ruby/Jekyll https://jekyllrb.com/docs/installation/windows/

```shell
gem install jekyll bundler
```

## Run
https://jekyllrb.com/docs/

```shell
bundle exec jekyll serve
```

Go to 
http://localhost:4000/
